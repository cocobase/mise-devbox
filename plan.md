## 🎯 核心达成指标 (Evaluation Metrics)
1. **启动性能**：执行 `mise run up` 后，**5秒内** 自动进入容器 Shell。
2. **环境就绪**：容器内 `python`, `node`, `uv`, `pnpm` 必须可用且版本与配置对齐。
3. **挂载验证**：宿主机当前目录必须实时同步至容器 `/workspace`。
4. **清理完整性**：执行 `mise run down` 后，`docker ps -a` 结果为空，且相关网络与进程无残留。
